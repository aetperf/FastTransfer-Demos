# Demos for FastTransfer

FastTransfer is a command line interface that allow to load data into a target table in a target database. Data can comes from another database or from files (CSV, parquet, json)

## Demos for MSSQL as target
[Copy from MSSQL (2022) to MSSQL (2025)](TargetMSSQL/FastTransferDemo1-MSSQL2022ToMSSQL2025.ipynb)

[Copy from PostgreSQL to MSSQL](TargetMSSQL/FastTransferDemo2-PGSQL2MSSQL.ipynb)

[Copy from Oracle to MSSQL](TargetMSSQL/FastTransferDemo3-Oracle2MSSQL.ipynb)

[Import from CSV or parquet to MSSQL](TargetMSSQL/FastTransferDemo4-Files2MSSQL.ipynb)

[Call FastTransfer from T-SQL using our free open-source T-SQL Wrapper](TargetMSSQL/FastTransferDemo5-TSQLWrapper.ipynb)
